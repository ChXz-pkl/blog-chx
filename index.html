Class dark: di Seluruh Elemen Setiap elemen kini memiliki dua set class. Contoh: bg-white dark:bg-gray-900. bg-white: Digunakan saat mode normal (light). dark:bg-gray-900: Digunakan saat class dark ada di tag . Logika JavaScript (di akhir ) Saat Halaman Dimuat: Script akan memeriksa apakah Anda pernah menyimpan pilihan tema di localStorage.
